## 📖 Overview
This lightweight, zero-config CLI tool scans target URLs to detect broken JSON-LD schema syntax that silently kills SEO rankings and blocks rich snippets. It aggressively extracts and validates markup blocks, identifying specific syntax errors, missing mandatory keys like @context and @type, and conflicting entity definitions without requiring API credits. Designed for SEO professionals and developers who need instant, mathematical precision over generic content analysis, it operates as a free and efficient alternative to heavy auditing platforms.
